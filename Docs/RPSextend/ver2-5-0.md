# ver2.5.0 — 封印／本命の分離とカード運用の訂正

- **日付**: 2026-07-08
- **種別**: 仕様訂正（モデル修正）。新規シミュレーションデータなし（再検証は latest §7-1 で継続）。

## 背景

ver2.4.0 までは「封印＝そのターンの本命の手」として記述していたが、実際のゲーム意図と食い違っていたため訂正する。封印と本命は別カードであり、読み合い（封印）と殴り合い（本命）を2系統に分ける。

## 訂正点

1. **封印 ≠ 本命**。封印は本命とは別に伏せる、相手の guess 対象になるリスク札。
2. **本命**は効果・能力カードを使用した後、手札から**両者同時に伏せる**（＝実際の戦闘手）。
3. 効果カード・能力カードは**毎ターン必須**（6ターンで各6枚を使い切る）。
4. 効果カードは伏せず**公開**して同時使用。
5. **guess は当否のみ判明**（相手の封印は覗けない）。
6. **封印はターン終わりに手札へ戻る**（消耗しない）。消耗するのは本命。
7. タイブレークが**全あいこ**なら引き分けで確定（追加フォールバックは設けない）。

## 手番順（確定）

封印（伏せ）→ 効果（公開）→ 能力（公開）→ 本命（伏せ）→ 公開（**本命のみ**）

## 推理モデル（確定）

封印は最後まで公開されず手札に戻るため、guesser は**公開された本命（公開情報）＋ show hand の開示**を追って、相手の残り手札＝封印候補を絞る。封印は必ず残り手札の1枚であり、floor2 により手札が2枚以上残る限り1枚に確定できない（命中の約1/3上限は据え置き）。

## 影響範囲

- **latest §2**：RPSカードの消耗ルール（本命＝消耗／封印＝手札へ戻る）、効果・能力カードの必須化・公開化。
- **latest §3**：手番順を封印→効果→能力→本命→公開に更新。公開は本命のみ。
- **latest §4**：推理対象を「公開された本命のトラッキング」に更新（floor2 均衡は維持）。
- **latest §5**：全あいこ＝引き分けで確定。
- **latest §6**：数値は訂正前モデルのもの。現行モデルでの**再検証が必要**（§7-1）。
- **latest §7**：再シミュレーション、および必須・公開カードによる情報開示のバランス影響を追加。

## 未反映（今後）

- 現行モデルでのモンテカルロ再実行によるスキル差・同型引分・封印guess命中率の測り直し。
- 「本命を伏せる前に効果カードの宣言手・使用能力が公開される」ことによる情報流の影響評価。
