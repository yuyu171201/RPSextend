# ver1.0.0 — 構造転換：6ターン固定・引き運ゼロ・RPS消耗

- 日付: 2026-07-08
- 種別: メジャー構造変更

## 変更内容
- **6ターン固定**、終了時の得点で勝敗（勝利点到達方式を廃止）。
- 特殊カード（効果・能力）を **最初から全所持・使ったら捨てる**（ドロー廃止＝引き運ゼロ）。
- RPSを **各3枚（計9枚）・使ったら捨てる**（消耗資源化）。

## 変更理由
- ver0.3.0 で判明した「ドロー運がスキルの天井を作る」問題を根本から解消するため。

## 検証データ
- 対称性：全同型50/50。
- expert vs beginner：**97.3%**（＝伏せ手＝本命を覗いて確定カウンターできるため決定的すぎ）。
- 引き分け：10〜19%（6ターン固定・小得点で多発）。

## 判明した課題
- 引き運を消すと、強いpeek（確定カウンター）が絶対化してスキル差が爆発（97%）。
- 「読み切れない封印」を保つ設計（ver1.1.0以降）が必要。
- 引き分け対策（サドンデス）が必要。
