# ver3.0.0 — オンライン対戦アプリの実装（CUI / 中央サーバー方式）

- **バージョン**: ver3.0.0
- **日付**: 2026-07-09
- **種別**: 実装マイルストーン（ルール変更なし。ver2.5.0 のルールをそのままアプリ化）
- **対象**: `Online/`（新規）

## 概要

これまで紙のカード（`cards.html`）＋ルールブック（`inst.md` / `latest.md`）で
遊ぶ形式だった「ジャンケン・エクステンド」を、**ターミナル上でオンライン1対1対戦
できるアプリケーション**として実装した。ルールは ver2.5.0（`latest.md`）に準拠し、
ルール自体の変更は行っていない。

設計方針は `Online/spec.md` に従う。

## 実装した内容

### アーキテクチャ（spec.md 準拠）
- **CUI**: ターミナルで動作するCLIアプリ（Python3・標準ライブラリのみ、依存なし）。
- **中央サーバー方式**: サーバーがゲームホスト（進行・判定役）。両プレイヤーはTCP
  ソケットで接続。
- **状態の一元管理**: 手札・封印・本命・得点・ターン進行の全状態をサーバーが保持。
  クライアントは選択肢の番号を送るだけで、盤面を書き換えられない（チート不可）。
- **情報の秘匿**: ハッシュ等の暗号は使わず、「両者の選択が揃うまで相手画面に出さない」
  という**表示制御**のみで同時出しの秘匿を実現（spec.md §2）。

### ステートマシン（spec.md §3）
サーバーが以下のフェイズを毎ターン遷移する。両者の入力が揃った段階で処理・通知する。

`MATCHING → 封印(伏せ) → 効果(公開) → 能力(公開/覗き見/guess/盾) →`
`本命(伏せ) → 公開・じゃんけん → guess判定 → 後始末` ×6ターン → 同点なら**サドンデス**。

### 再現したルール要素（ver2.5.0）
- **封印≠本命**の分離：封印はターン終わりに手札へ戻る（消耗しない）、本命は捨てる。
- 効果カード（公開・毎ターン必須）と**効果ボーナス**（宣言手と勝った本命の一致で+1）。
- 能力カード（毎ターン必須）：
  - **覗き見(show hand)**：相手が1枚開示→自分が最大2枚指定（**floor2**：非公開2枚を常に残す）。
  - **guess(手)** +1 / **guess(手+色)** +2（**1〜3ターンは+3**）。**当否のみ判明**。
  - **盾(guard)**：相手のguessを無効化＋自分+1。
- **サドンデス**：残りRPS（各自3枚）を封印なしで出し合い、先に1勝で決着。全あいこは引き分け。
- 開示の**上下マーカー**（直前ターンに開示された札かを盤面に `*開示中*` として表示）。

## ファイル構成（`Online/`）

| ファイル | 役割 |
|---|---|
| `server.py`   | 中央サーバー。マッチング・ステートマシン・判定・盤面描画 |
| `client.py`   | プレイヤー用CUIクライアント（番号入力で操作） |
| `game.py`     | カード定義・`GameState`・勝敗/得点ロジック |
| `protocol.py` | 改行区切りJSON（NDJSON）の送受信ユーティリティ |
| `README.md`   | 起動・接続方法 |
| `spec.md`     | オンライン対戦仕様（既存） |

## 通信プロトコル（NDJSON）
1行1メッセージのJSON。
- サーバー→クライアント：`HELLO` / `MSG` / `STATE`（自分視点の盤面） / `PROMPT`（入力要求） / `GAMEOVER`
- クライアント→サーバー：`NAME` / `CHOICE`（PROMPTへの回答）

## 動作検証
- 乱数入力ボットによる複数フルゲーム完走（通常決着・サドンデス決着の両方）を確認。
- 封印/本命の秘匿、効果ボーナス、覗き見(floor2)、guess当否、盾による無効化、
  サドンデスの分岐、相手切断時の通知、連続マッチングを確認。サーバー側の例外なし。

## 起動方法（要約）
```bash
# サーバー
cd Online && python3 server.py --port 5000
# クライアント（2人）
python3 client.py --host <サーバーIP> --port 5000 --name アリス
python3 client.py --host <サーバーIP> --port 5000 --name ボブ
```
