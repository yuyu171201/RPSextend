- 日付: 2026-07-08
- 種別: 原案

## 仕様（この版）
- RPS グー/チョキ/パー 各1枚（毎ラウンド手札に戻る）。
- 能力カード・追加効果カードを各種2枚、別々の山札にシャッフルしてドロー。
- 毎ターン手札3枚に補充 → 手順2で1枚を伏せ（囮）→ 能力/効果使用 → 残り2枚から本命 → 勝敗、勝ち+1 → 勝利点到達まで。

## 変更理由
- ユーザー原案の初期評価。

## 検証データ
- ランダム同型：**49.5 / 50.5%**、平均16.5ラウンド（勝利点7）→ 三すくみは対称で健全。
- しきい値とゲーム長：5点=11.3R / 7点=16.7R / 10点=24.8R。
- greedy(効果荒稼ぎ) vs random：59.6%（ただし counter に37.9%で狩られる）。

## 主な指摘
- 手順1「手札3枚に補充」でRPSが戻る前提が未定義（明文化要）。
- 手順2の囮の価値が薄い／「伏せカードを見る」能力がほぼ無意味。
- 能力カード3種の強弱が不均衡（能動偵察 >> 受動公開 >> 伏せ見）。
