# ver2.4.0 — show hand color 削除・guard guess 追加【現行】

- 日付: 2026-07-08
- 種別: カード構成変更（現行仕様）

## 変更内容
- **show hand color ×2 を削除**。
- 空いた枠に **show hand ×1** と **guard guess ×1** を追加。
- 結果、能力6枚 = **show hand ×3 / guess(+1) ×1 / guess色(+2) ×1 / guard guess ×1**。

### guard guess
- そのターン相手が自分の封印を guess してきたら、**その guess を無効化＋自分に +1**。相手が撃たなければ空振り。

## 変更理由
- **show hand color は show hand と実質同じ**（開示すれば手も色も判明するため色専用は冗長）。
- 冗長枠を、guessを狩る **guard** に置き換え、推理レイヤーに「撃つ vs 撃ち手を罰する」読み合いを追加。

## 検証データ
- guardなし expert vs beginner：64.7%。
- **guardあり expert vs beginner：69.6%**（予測しやすい初心者の撃ち時を狩って +5pt）。
- 同型引き分け：24.7%（*決定的モデルの過大評価。混合タイミングで低下見込み）。

## 意義
- guardは「相手の撃ち時を読む」実力を報酬化し、guess連打を抑制。1枚で適量。
- 冗長を削り読み合いを足す、役割の被らない良い再構成。
