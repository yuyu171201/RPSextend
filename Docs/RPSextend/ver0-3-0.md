# ver0.3.0 — 固定デッキ（ドロー運あり）でスキル差測定

- 日付: 2026-07-08
- 種別: 検証（ドロー運の上限を測る）

## 仕様（この版）
- 効果カード6枚（各手×{+1,+2}）＋能力カード6枚（show hand×3, show hidden×2, guess+1×1）をデッキからドロー。
- 勝利点しきい値に到達で終了。

## 変更理由
- 具体的なカード構成で、上級者 vs 初心者のスキル差が目標(80%)に届くか確認。

## 検証データ
- 対称性：全同型50/50。
- expert vs beginner：th7=61% / th12=64% / th18=67% / th25=70%（**約70%で頭打ち**）。
- 原因：情報カードを引けないターンが半分あり、1ラウンドあたりの優位が頭打ち＝**ドロー運がスキルの天井を作る**。

## 決定
- 引き運を消す構造転換（ver1.0.0）へ進む動機付けとなった。
