# ver2.2.0 — 上下マーカー（開示の1ターン持続）と最適封印の是正

- 日付: 2026-07-08
- 種別: メカニクス追加＋モデル修正

## 変更内容
- カードに **上下（向き）** を付与。直前ターンに開示された札かを判別可能に。
- 効果：相手が「開示済み札を封印したか」が分かる → 封印候補を「開示済みプール／非開示プール」に分けて推理できる。

## 検証データ（重要な是正あり）
- 素朴モデル（封印側が開示札を絶対伏せない）：expert vs beginner **73〜77%**。ただし **同型引き分け53%** に爆発。
- **是正**：封印側は「開示3枚のプールに隠れれば約1/3で守れる」ため最適には**大きい方のプールに隠れる**。ユーザー指摘により再測定：
  - expert vs beginner：**63.8%**（報酬3で67.1%）。
  - **同型引き分け：53% → 17.6% に激減**（混合戦略の揺らぎが生まれ、決定的な一致が消えた）。

## 変更理由・意義
- マーカーは「プール当て」の新しい読み合いを生み、**開示札をあえて封印するブラフ**が有効手に。
- floor（読み切れない封印）を壊さずに読み合いを深化。過大評価も是正され引き分け問題も改善。
