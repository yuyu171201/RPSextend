# ver2.3.0 — 早撃ち補償（1〜3ターンの色guess +3）

- 日付: 2026-07-08
- 種別: 数値調整

## 変更内容
- **1〜3ターンのみ、色guess（手＋色）の報酬を +2 → +3** に増加。手guessは +1 のまま据え置き。

## 変更理由
- 素の報酬だと序盤は候補プールが大きく命中が低いので「常に終盤guessが正解」で単調化（when-to-guessが解けてしまう）。
- 序盤の低命中を報酬で補償し、早撃ちEVを終盤と釣り合わせて**真の選択**にするため。

## 検証データ（基準：終盤 guess 平均1.60点）
- 序盤 +1/+2（素）：1.31点（早撃ち損＝単調）。
- 序盤 **+1/+3**：**1.68点 ≒ 終盤1.60**、expert 63.9%、引分8.0%（★釣り合う）。
- 序盤 +2/+3：2.23点（過剰）。
- 序盤 +2/+4（当初案）：2.60点・引分13.6%（過剰＋高分散）。

## 発見
- ブーストが必要なのは **色guessだけ**。手guessは3値しかなく序盤でも多数派読みで命中が落ちにくいため据え置きでよい。
