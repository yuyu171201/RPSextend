# ver2.1.0 — 有償・部分開示（floor 2）とサドンデスTB

- 日付: 2026-07-08
- 種別: メカニクス追加

## 変更内容
- **開示ルール**：相手が1枚（相手選択）＋自分が2枚（自分選択）＝相手の札を最大3枚見る。自分は漏らさない。**相手の非公開札は常に2枚残す（floor 2）**。順序は相手優先。
- **タイブレーク**：6ターン同点時、残RPS（各3枚）を封印なしで1枚ずつ出し、先に勝ったラウンドを取った方の勝ち。

## 変更理由
- floor 2 により **封印の推測は約1/3が上限**（封印＋隠し2枚＝3候補以上）＝solve不能を数理的に保証。「読み切れない封印」を実現。

## 検証データ
- 推理のみ：expert vs beginner **48.3%**（floorが強すぎて読みが弱い）。
- **戦闘の残り手読み**（捨て札トラッキング）を追加：**59.8%**（floorを保ったままスキル差）。
- タイブレーク：公平48/48、96%が3R以内決着、全あいこ4%。

## 判明した課題
- floor 2 と読みの強さは綱引き。floor維持だと天井は約60〜70%。
- 開示の持続効果（ver2.2.0のマーカー）で読みを補強する余地。
