# ver4.2.0 — 盤面を直接クリックして操作（開示は上下反転で表示）

- **バージョン**: ver4.2.0
- **日付**: 2026-07-12
- **種別**: 実装（ルール変更なし。ブラウザGUI版の入力UIを刷新）
- **対象**: `Online/web_server.py`, `Online/engine.py`

## 概要

ver4.1.0 では「操作パネルに並んだカードの選択肢」をクリックする方式だったが、本バージョンで
**盤面の自分のカードそのものを直接クリックして操作する**方式に変更した。
例えば「見せる手札を選択」では、盤面右の **あなた** ゾーンの手札だけが光ってクリック可能になり、
選ぶと公開される。さらに、**開示された札はゲームの上下マーカー通り 180°上下反転**で表示する。

盤面に対応する札が無い選択（guess の手・色の予想、覗き見での相手札の指定）だけは、従来どおり
操作パネルにカードの選択肢を出す。

ゲームのルール・判定ロジック（ver2.5.0）は不変。

## 変更点

### 1. サーバー: プロンプトに「選択元ゾーン(source)」を付与（`engine.py`）

- `Session.prompt(pid, text, options, kind=None, source=None)` に `source` を追加し、`PROMPT`
  に同梱。選択肢が盤面のどの札に対応するかを示す:
  - `rps_hand` … 自分のRPS手札（封印／本命／覗き見の自己開示／サドンデス）
  - `effect`   … 自分の効果カード
  - `ability`  … 自分の能力カード
  - `panel`（既定）… 盤面に無い選択（guess の手/色、覗き見の相手札指定）。従来のパネル選択。
- 各フェイズの `prompt_both` / `prompt_one` 呼び出しに `source` を指定
  （`specs` を 5 要素タプル `(pid, text, options, kind, source)` に）。

### 2. クライアント: 盤面クリック操作＋開示の上下反転（`web_server.py`）

- `PROMPT` の `source` が `rps_hand`/`effect`/`ability` の場合:
  - 操作パネルには操作説明＋ヒント（「盤面右の“あなた”ゾーンで光っているカードをクリック」）
    のみを表示。
  - 盤面を再描画し、**該当ゾーンの選択可能な札だけを発光（`.sel`, パルス）**、
    そのゾーンの他の札は淡色化して選択対象を明確化。
  - カードに `data-key` を持たせ、`#board` へのイベント委譲でクリック→選択を確定。
- `source` が `panel` の場合は従来どおり操作パネルにカードの選択肢を出す
  （guess の手＝手カード、色＝色スウォッチ、覗き見の相手札指定＝RPSカード）。
- **開示済みマーカーを上下反転に変更**: `revealed` な RPS カードを `transform: rotate(180deg)`
  で 180°反転表示（ゲームの「上下の向き＝直前ターンに開示された札」の再現）。発光リングも付与。
  相手ゾーンの「開示された札」も同様に反転表示。
- 状態管理: 直近盤面 `lastView` と選択中プロンプト `activePrompt` を保持し、
  盤面クリックとカード発光の再描画を一貫させる。キーボード操作（Enter/Space）にも対応。

## 変わっていないもの
- ルール（ver2.5.0）、勝敗・guess/guard・サドンデスの判定ロジック。
- 秘匿方式（相手の非公開情報はサーバーが送らない）。
- 既存エンドポイント `/`, `/join`, `/poll`, `/choice`, `/leave`, `/admin/status` の挙動。
- 盤面表示のカードデザイン（ver4.0.0）・実況ログ・ルール説明・切断時の自動終了。
- CUI 版（`server.py` + `client.py`）。

## 動作検証
- ブラウザプレビューで2人をマッチさせ、1ターンの全フェイズを盤面クリックで操作:
  - 封印／効果／能力／本命: 盤面の該当ゾーンの札だけが発光し、他ゾーンは淡色化。
    クリックで選択→送信され、次フェイズへ進むことを確認。
  - 覗き見の対象になった側: 盤面の自分の手札から開示する札をクリックして選択。
  - 開示後の再描画で、開示された3枚（自己開示1＋相手指定2）が**上下反転**表示されることを確認
    （実況ログの「あなたのカードが開示されました …」と一致）。
  - guess の手／色は操作パネルのカード（手カード／色スウォッチ）で選択。
  - 反転かつ選択可能な札のクリックも正常動作。ターンが正しく進行。
- コンソールエラーなし。
