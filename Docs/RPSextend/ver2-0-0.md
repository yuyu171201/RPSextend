# ver2.0.0 — 色概念の導入と「封印＝リスク」

- 日付: 2026-07-08
- 種別: メジャー機能追加

## 変更内容
- RPSに **色（R/G/B）** を付与：`手 × 色` の9枚すべてユニーク。色は戦闘に無関係、推理の第2軸。
- 封印を **完全なリスク** 化：相手に伏せ手を当てられると相手が得点。
- guess を2レーン化：**手を当てて +1 / 手＋色を当てて +2**。

## 変更理由
- 「show hidden と封印の価値が薄い」という指摘。封印を読まれ得点対象にして毎ターンのリスクにし、色で多角推理を可能にするため。

## 検証データ
- 対称性：全同型ほぼ50/50。
- expert vs beginner：**68.1%**。
- 引き分け：6〜19%。

## 判明した課題
- この時点のモデルは「開示＝直接開示」で、show→guessが確定コンボ化し上級者同型が横並び（solvableで天井が浅い）。
- → 有償・部分開示（ver2.1.0）で「多角的推測」を本物にする必要。
