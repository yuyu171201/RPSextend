# ver0.2.0 — 効果宣言・推理カードのバランス調整

- 日付: 2026-07-08
- 種別: バランス調整（後の構造転換で多くが置換される探索フェーズ）

## 変更内容
- 効果カードの運用を「強制・公開宣言」ではなく「任意・伏せ」に。
- 予言（相手の伏せ手を当てる）能力の **+1点を削除し「情報のみ」** に。

## 変更理由
- 強制公開宣言だと、宣言通り出す honest が、宣言を狩る reader に一方的に負ける（後出し有利は報酬では直らない）。
- 予言に得点を付けると運ゲー化・即決化する。

## 検証データ
- 強制公開・honest vs reader：**18.9 / 81.1%**（均衡混合bluffだと reader と五分に是正）。
- 予言+1あり：毎戦6〜8回発動・2点前後が33%の運で入り、ゲームが4〜9Rで即決。
- 予言を情報のみに：17R前後の健全な長さに戻り、使う側が56〜64%（skill toolとして機能）。

## 決定
- 効果は任意・伏せ、予言は情報化の方向を採用（この後の色付き設計に継承）。
