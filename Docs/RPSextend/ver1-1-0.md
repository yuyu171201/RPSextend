# ver1.1.0 — 封印を手札に戻す（囮の意味を回復）

- 日付: 2026-07-08
- 種別: メカニクス修正

## 変更内容
- 封印（囮）は伏せるが **ターン後に手札へ戻す**。実際に出した本命だけを捨てる。

## 変更理由
- ver1.0.0 で「伏せ手＝本命」だと covpeek が確定カウンター化して97%に。
- 消耗資源下では囮に1枚使うのは損（リスク>価値）、行き先も未定義という指摘。
- 囮を戻すことで低コスト化し、本命は残り2枚から選ぶ形に戻る＝**peekは囮を見る＝「負けない手」止まり**になり過剰なスキル差が緩和。

## 検証データ
- expert vs beginner：**64.4%**（97%から是正）。
- 対称性維持、引き分け約11%。

## 決定
- 封印＝「敵のpeekを空振りさせる安価な盾」として定義。以降の色付き設計の土台に。
