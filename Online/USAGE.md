# RPS Extend Online - Usage Guide

このディレクトリ (`Online/`) には、ジャンケン・エクステンドのオンライン対戦用スクリプトが含まれています。
先ほど構築した `uv` のPython環境を利用して、安全かつ確実に実行する方法を説明します。

---

## 準備

ターミナルを開き、`Online` ディレクトリに移動します。

```bash
cd Online
```

---

## 1. サーバーの起動

ゲームの進行や判定を行う中央サーバーを起動します。どれか1台のPC（またはクラウドサーバー）で実行してください。

```bash
uv run server.py
```
*(デフォルトでは `0.0.0.0:5000` で接続を待ち受けます)*

特定のIPやポートを指定したい場合は、以下のようにオプションを付けます。
```bash
uv run server.py --host 127.0.0.1 --port 8080
```

---

## 2. クライアント（プレイヤー）の接続

対戦を行うプレイヤーは、クライアントスクリプトを起動してサーバーに接続します。
対戦を始めるには、2人が接続する必要があります。（同じPCでテストする場合は、ターミナルを2つ開いて実行します）

**プレイヤー1の接続例:**
```bash
uv run client.py --host 127.0.0.1 --port 5000 --name アリス
```

**プレイヤー2の接続例:**
```bash
uv run client.py --host 127.0.0.1 --port 5000 --name ボブ
```

*※別のPCから接続する場合は、`--host` にサーバーを起動しているPCのローカルIPアドレス（またはグローバルIP）を指定してください。*

---

## 3. ゲームのプレイ

2人が接続すると自動的に対戦が開始されます。
画面（CUI）の指示に従い、自分のターンが来たら選択肢の **番号（数字）を入力** して Enter を押すだけでゲームが進行します。

**入力例:**
```text
封印するRPSカードを選択:
  1) グー(赤)
  2) チョキ(緑)
  ...
> 1
```

*※3人目以降が接続した場合は待機列に入り、現在の対戦が終了次第、次のプレイヤー同士が自動でマッチングされます。*
