# RPS Extend (ジャンケン・エクステンド)

じゃんけん（グー・チョキ・パー）に情報戦とブラフ、資源管理の要素を加えた、運要素ゼロの2人対戦型カードゲームです。

このゲームは既にプレイ可能な状態になっています。実際に遊ぶ際は以下の2つのファイルのみを使用してください。（その他の開発用ファイル等を見る必要はありません）

## 遊び方・必要なもの

1. **ルールの確認**
   [**inst.pdf**](./inst.pdf) をご覧ください。ゲームの詳しい進行や勝敗条件が記載されています。（[**inst.md**](./inst.md) でも同じ内容をテキストで確認できます）

2. **カードの準備**
   [**cards.html**](./cards.html) をブラウザで開き、A4用紙等にカラー印刷してください。（1ページ目がRPSカード、2ページ目が効果カード、3ページ目が能力カードになっています）。
   印刷したカードを線に沿って切り取ることで、1人分の対戦カードになります。

## 概要

* **プレイ人数**: 2人
* **プレイ時間**: 約10〜15分
* **対象年齢**: 10歳以上

両プレイヤーは全く同じ構成のカード（手札）を持ってゲームを開始します。
相手の出したカードを記憶し、残りの手札を推理して、相手の行動を読み切った者が勝利します！
